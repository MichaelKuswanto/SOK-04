Static Website and Dynamic Website


Created on 09 Mar 2023

Static Website

Situs web yang tidak berubah berdasarkan interaksi pengguna; biasanya dibuat menggunakan HyperText Markup Language (HTML) dan Cascading Style Sheets (CSS)


Dynamic Website

Situs web yang berubah berdasarkan interaksi pengguna; sering kali dibangun menggunakan Python, JavaScript, PHP, atau ASP dengan HTML
